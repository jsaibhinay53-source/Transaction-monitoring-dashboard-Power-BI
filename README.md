# Transaction-monitoring-dashboard-Power-BI
Built an AML Transaction Monitoring Dashboard using Power BI and Excel to analyze 15,000+ transactions, monitor suspicious activities, track investigation alerts, and identify high-risk customers through interactive visualizations and DAX-based KPIs.

**Project Overview**

The Transaction Monitoring Dashboard is a Financial Crime Analytics project developed using Power BI, Excel, and DAX. The objective of this project is to monitor customer transactions, identify suspicious activities, track compliance alerts, and provide actionable insights for Anti-Money Laundering (AML) operations.

This dashboard helps compliance teams and financial institutions detect unusual transaction patterns, monitor high-risk customers, and improve investigation efficiency through interactive visualizations and real-time reporting.

**Business Problem**

Financial institutions process thousands of transactions daily. Monitoring these transactions manually is time-consuming and increases the risk of overlooking suspicious activities.

The goal of this dashboard is to:

Monitor large volumes of financial transactions.
Detect potentially suspicious transactions.
Identify high-risk customers.
Track AML investigation alerts.
Improve compliance reporting and decision-making.
